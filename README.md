# 🍛 Asia Food
Página para o restaurante *Asia Foods*, especializada em comida asiática (japonesa, coreana e chinesa), com um toque brasileiro. O projeto apresenta o restaurante, exibindo o seu cardápio, tendo a possibilidade de realizar reservas pelo próprio site.

## Sobre o Projeto
O **Asia Foods** é um projeto esenvolvido em HTML5 e CSS3.

## Funcionalidades
* Barra de navegação fixa com com acesso a todas as sessões
* Formulário de reserva de mesas(solicitando nome, email, celular, data, horário,número de pessoas e qual tipo de ambiente)
* Cardápio com fotos dos pratos
* Área de contato com localização, telefone e email
* Efeito hover e placeholder em toda página
